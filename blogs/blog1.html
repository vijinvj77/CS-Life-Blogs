Introduction to Java

Today I started my course on UPgrad on Java. What I learnt in the first session was introduction on Java. 
         In today’s session, I learnt about JDK and how to install it, components of JDK, how code written in Java gets converted into machine code, etc.  
         JDK stands for Java Development Kit. JDK contains the JRE and a Java compiler. The Java compiler converts the source code to a byte code.
         This byte code gets converted to the final machine code by the JVM(Java Virtual Machine), which contained by the JRE(Java Runtime Environment). 
         Computer understands everything in machine language, and all the data on a computer stores as binary bits, i.e. 0 and 1.
